# Trunk-Mode-cisco-packet-tracer-project
Overview

A collection of networking mini projects created using Cisco Packet Tracer to practice network configuration, VLAN setup, trunk mode configuration, and troubleshooting.

Features
VLAN configuration
Trunk mode setup
Router and switch configuration
IP addressing and subnetting
Network troubleshooting
Technologies Used
Cisco Packet Tracer
Networking Fundamentals
Included Projects
VLAN Configuration Project
Trunk Mode Configuration Project
Basic Router and Switch Setup
Small Office Network Simulation
